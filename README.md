# Spectral Clustering Project Bundle

This bundle contains a LaTeX report, figures, CSV results, and a runnable Python codebase for the **Spectral Clustering** final-project track.

## Contents
- `main.tex` — report
- `refs.bib` — bibliography
- `final_pic/` — all figures referenced by the report
- `results/` — CSVs and JSON summaries generated by the experiments
- `code/` — runnable Python workspace
- `TODO.md` — status against the original PDF brief
- `PLAN.md` — extension ideas for future work / Claude Code handoff

## Quick start
```bash
cd code
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
OMP_NUM_THREADS=1 OPENBLAS_NUM_THREADS=1 MKL_NUM_THREADS=1 python run_all.py
```

That command regenerates:
- figures in `../final_pic/`
- result tables in `../results/`

## What the code currently runs
- Synthetic success cases: two moons, circles, blobs, high-dimensional moons, high-dimensional blobs
- Synthetic failure/stress cases: overlapping blobs, bridge-connected clusters
- Real/vector data: Iris, digits subsets `{0,1}`, `{1,7}`, `{0,1,3,6}`, and all digits
- Graph data: karate club and a stochastic block model
- Robustness: additive Gaussian noise on two moons
- Scaling: runtime and approximate memory versus sample size

## Reproducibility notes
- The code is deterministic under a fixed random seed.
- The main experiment entrypoint is `run_all.py`.
- The implementation is intentionally lightweight and offline-runnable.
